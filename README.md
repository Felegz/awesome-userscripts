1. Create a GitHub repo named `awesome-userscripts` (via website or `gh` CLI).

```

git push -u origin main
```

Or with the GitHub CLI (if authenticated):

```
gh repo create <your-username>/awesome-userscripts --public --source=. --remote=origin --push
```
